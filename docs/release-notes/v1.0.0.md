# FreeLook 1.0.0

FreeLook 1.0 brings a polished Quick Look experience for developer files on macOS.

## New in 1.0

- Quick Look previews for a broad set of developer-oriented file types
- Shiki-based syntax highlighting with curated light and dark themes
- Markdown rendering with GitHub-flavored Markdown support
- JSON pretty-printing and syntax highlighting
- XML pretty-printing and syntax highlighting
- Support for property lists and entitlement files
- Shared preview preferences through the host app:
  - Theme mode
  - Light theme
  - Dark theme
  - Code font
  - Code font size
- Graceful fallback behavior for:
  - large files
  - invalid JSON
  - invalid XML
  - binary files

## File Types Covered

FreeLook 1.0 covers:

- Markdown, JSON, XML
- Property lists and entitlement files
- Swift, JavaScript, TypeScript, Python, shell, Ruby, HTML, CSS
- Rust, Go, Java, Haskell, Clojure
- Kotlin, C#, PHP, Lua, Scala
- YAML, TOML, SQL
- `Dockerfile` and `Makefile`

## Notes

- Generic plain text (`.txt`) is not treated as a guaranteed FreeLook surface because the system plain-text Quick Look path often takes precedence.
- Some files may not route to FreeLook on machines where Launch Services still resolves it to other third-party metadata. It's beyond the scope of FreeLook.
